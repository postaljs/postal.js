# postal

Pub/Sub message bus for JavaScript and TypeScript. Wildcard subscriptions, channel-scoped messaging, and zero dependencies.

> **v3.0.0** — Full TypeScript rewrite. This is a work in progress.

## Install

```bash
npm install postal
```

## Status

This package is under active development. The v2.x API surface is being ported to TypeScript.

See the [root README](../../README.md) for development instructions.
